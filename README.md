# JMeter RPH Calculator Plugin

![Plugin Interface](img_3.png)

Этот плагин избавляет от необходимости вручную высчитывать количество потоков и параметры таймеров. Вы задаете целевой RPH (запросов в час) и длительность скрипта, а плагин сам настраивает Thread Group и Pacing. <br>

Выставляете: 
-target rph (например 1050) <br>
-iteration duration (общее время выполнения всех сэмплеров в тред группе, например 3 секунды) <br>
-ramp-up (время, за которое поднимутся все треды) <br>
-hold (длительность нагрузки(теста)) <br>
-ramp-down (время завершения работы тредов) <br>
-HTTP Samplers (количество сэмплеров в тред группе, сканируется автоматически, но можно указать вручную) <br>
-Threads (количество тредов, выставляется автоматически, но можно менять вручную) <br>

Формула: <br>
actual rph / 60 / samplers = target throughput <br>
1050/60/11 = 1,59090909 -> это значение устанавливается в constant throughput timer, который внутри flow control action <br>

После расчета автоматически добавляется flow control action внутри которого constant throughput timer, а так же User Defined Variables, в котором сохраняются значения из таблицы
